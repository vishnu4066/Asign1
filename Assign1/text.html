Full stack development with Django

This weekend at VIT CSE Department. We would be discussing following topics in order.

  1. Basics of Internet
  2. Web App Stack
  3. HTML
  4. CSS with Bootstrap
  5. JavaScript with jQuery
  6. AJAX
  7. Django and MySQL

We would like the students to have following traits.

  • Decent knowledge of python and basic coding skills
  • Be enthusiastic
  • Be willing to innovate
  • Be eager to learn by exploring and asking good questions
  • Be willing to work hard

And finally not complain about anything. There will be snacks and beautiful rewards.
